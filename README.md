# ScrapboxHelp

- [OmniHelp](https://github.com/NEKN3X/OmniHelp)のOmniboxを使わないversion
- 主な機能
  - Scrapboxのページをあいまい検索する
  - Scrapbox以外のWebページをあいまい検索する
- Webページを保存する際にHelpfeel記法でヘルプを書く
  - xyz(に関する(情報|記述)|のドキュメント)
  - https://example.com/xyz
